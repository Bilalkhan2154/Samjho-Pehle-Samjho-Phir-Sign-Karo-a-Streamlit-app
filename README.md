🪶 Samjho — Pehle Samjho, Phir Sign Karo
AI-powered legal document assistant for empowering India's underserved millions.

🌟 Features
Multi-format document intake (PDF, DOCX, images with OCR)
Intelligent document classification (Employment, Loan, Rental, NDA, etc.)
Tiered risk warnings (Critical/Caution/Info)
Multilingual explanations (12+ Indian languages)
Voice-assisted form filling (experimental)
What-If scenario simulator
Evidence PDF report generation
Post-submission lifecycle support (reminders, exit strategies)
